# GRACIA — AI Command Centre

A cinematic dark landing page for GRACIA, an AI launch platform ("from idea to live app").

Single self-contained HTML file — no build step, no dependencies. Just open `index.html`.

## Features
- "Lights bloom on" reveal intro with an animated GRACIA loading wordmark
- A real laptop composited onto a dark studio-office scene
- Serif "writing on the wall" headline set inside the room's framed art panel
- Click the laptop screen to zoom into a fully interactive dashboard (8 tabs)
- Film grain, headline light-sweep, and a custom magnetic cursor (desktop)
- Fully responsive

## Run locally
Open `index.html` in any browser, or serve it:
```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy
Works as-is on GitHub Pages, Netlify, Vercel, or any static host — it's one file.
